## What is included?

- Standard config base;
- React hooks plugin;
- React plugin;
- Prettier;
- JSX a11y plugin;

## How to use

1. Install dependency

```bash
yarn add eslint @oigabrielteodoro/eslint-config -D
```

2. Add `extends` in your file eslint config (Ex: `.eslintrc.json`)

```json
{
  "extends": "@oigabrielteodoro/eslint-config/react"
}
```
